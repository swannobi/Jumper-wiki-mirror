### Command List

| Commands 	| Description                                       	|
|----------	|---------------------------------------------------	|
| start    	| Starts a raffle                                   	|
| buy      	| Buys raffle ticket(s)                             	|
| check    	| Shows you the number of raffle tickets you bought 	|
| cost     	| Sets the cost of raffle tickets                   	|
| end      	| Ends a raffle                                     	|
